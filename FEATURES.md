# 功能说明

## ✨ 核心功能

### 1. 实时汇率计算
- 集成 exchangerate-api.com 免费 API
- 汇率数据自动缓存 1 小时，减少 API 调用
- 支持 9 种主流货币：
  - 💵 USD (美元)
  - 💶 EUR (欧元)
  - 💷 GBP (英镑)
  - 💴 CNY (人民币)
  - 💴 JPY (日元)
  - 💵 HKD (港币)
  - 💵 SGD (新加坡元)
  - 💵 AUD (澳元)
  - 💵 CAD (加元)

### 2. VPS 价值计算（全新升级 🎉）
输入以下信息即可计算：
- **购鸡金额（CNY）**：首次购买价格
- **续费金额（CNY）**：续费价格（可能与首次不同）
- **货币类型**：从 9 种货币中选择
- **今日汇率**：实时显示当前汇率（只读，自动更新）
- **付款周期**：月付、季付、半年付、年付
- **购买日期**：使用日期选择器选择（默认今天）
- **到期时间**：使用日期选择器选择

**智能日期计算**：
- 自动计算总天数 = 到期时间 - 购买日期
- 自动计算已使用天数 = 今天 - 购买日期
- 自动计算剩余天数 = 到期时间 - 今天

### 3. 详细统计显示

#### 📊 使用进度
- 可视化进度条显示使用百分比
- 显示已使用天数 / 总天数

#### 💰 财务统计
- **每日成本**：平均每天花费多少
- **已消耗价值**：到目前为止花费的金额
- **剩余价值**：还剩多少价值（高亮显示）
- **剩余天数**：还能使用多少天

#### 🌍 多币种显示
自动将剩余价值转换为其他 8 种货币显示，方便对比

#### 🔄 续费建议（新功能）
智能对比续费成本：
- **续费每日成本**：续费金额 / 付款周期天数
- **成本对比**：续费与首购的成本差异百分比
- **智能建议**：
  - ⚠️ 续费成本上涨 X%（橙色警告）
  - ✅ 续费优惠 X%（绿色提示）
  - ℹ️ 续费成本相同（灰色提示）

## 🚀 快速开始

1. 打开浏览器访问: http://localhost:3457
2. 输入 VPS 购买信息
3. 点击"计算剩余价值"
4. 查看详细的计算结果和多币种对比

## 💡 使用技巧

- 支持键盘 Enter 快捷计算
- 汇率每小时自动更新
- 可以随时切换货币重新计算
- 多币种显示会自动排除输入货币本身
